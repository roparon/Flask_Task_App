Perfectly Styled Instruction Guide

Using HTML, Bootstrap, CSS Variables, and Inline Styles

Step 1: Project Setup

  • Install a code editor (e.g., VS Code).
  • Create a new folder for your project files.
  • Inside it, create an index.html file.
  • Set up your HTML structure with <!DOCTYPE html>.

Step 2: Adding HTML Content

  • Use semantic tags like <header>, <main>, <footer>.
  • Structure content using <div>, <section>, and <ul>.
  • Use <h1> to <h6> for headings and <p> for paragraphs.

Step 3: Styling

  • Use Bootstrap classes like .container, .btn, and .list-group.
  • Define your own variables with :root and use them throughout your CSS.
  • Add custom styles in a <style> block in the head.
  • Use inline styles for small, one-time tweaks like <div style="margin-top:20px;">.
