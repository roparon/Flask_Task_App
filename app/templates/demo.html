How to Build a Simple Website

Your step-by-step instruction guide

Basic Steps

  • Install a code editor (e.g., VS Code, Sublime Text).
  • Create a new folder for your project.
  • Inside the folder, create an index.html file.
  • Add your HTML structure using <!DOCTYPE html>, <html>, <head>, and <body> tags.
  • Use <h1>, <p>, and <div> tags to add content.

Styling with CSS

  • Use a <style> tag in the <head> to write CSS.
  • Use selectors like body, h1, or classes like .container.
  • Customize fonts with font-family and colors with color and background-color.
  • Use margin, padding, and box-shadow to add spacing and depth.

Next Steps

  • Add more pages and link them with <a href="page2.html">.
  • Use images with the <img src="image.jpg"> tag.
  • Learn responsive design with media queries.
  • Explore JavaScript to make your site interactive.
