How to Build a Simple Website

Your step-by-step instruction guide

Basic Steps

  • Install a code editor (e.g., VS Code, Sublime Text).
  • Create a new folder for your project.
  • Inside the folder, create an index.html file.
  • Add your HTML structure using <!DOCTYPE html>, <html>, <head>, and <body> tags.
  • Use <h1>, <p>, and <div> tags to add content.

Adding Styles

To style your website, you can use CSS. Here are some basic steps:

  • Create a new file named styles.css.
  • Link the styles.css file in your index.html using the <link> tag.
  • Add CSS rules to style your HTML elements.

Example CSS:

        
          body {
            background-color: #f8f9fa;
          }
          h1 {
            color: #0d6efd;
          }
